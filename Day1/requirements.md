Project: MOBZ – Mobile Recharge Web Application
1. Project Overview
A simple and attractive mobile recharge web application where users can browse recharge plans, view operators, and simulate a recharge flow using only HTML, Tailwind CSS, and JavaScript.

Enter number

Select operator

Select plan

3. UI/UX Requirements
Clean, polished layout
